## Overview

The T-PAK Dormitory Matching Platform is a web application created to streamline the process of finding suitable dormitory accommodation across Thailand. With a user-friendly interface and powerful matching algorithms, the platform helps users easily locate dorms that meet their requirements based on factors such as location, price, and amenities.

The platform was built with a focus on efficiency and ease of use, utilizing a modern tech stack that includes React for the frontend, Node.js and Express for the backend, and MongoDB for scalable data storage.

## Key Features

- **User Matching Algorithm**: Implemented a robust matching algorithm to pair users with dormitories that match their specified criteria, including budget, location, and amenities.
- **React Frontend**: Developed an intuitive, responsive frontend interface using React, allowing users to search, filter, and view details of available dorms across various regions in Thailand.
- **Node.js and Express Backend**: Built a secure, scalable backend using Node.js and Express to handle API requests, manage user data, and connect with MongoDB for data retrieval and storage.
- **MongoDB Database**: Utilized MongoDB for flexible data storage, enabling efficient data management for listings, user profiles, and booking information.

## Challenges and Learnings

- **Efficient Data Retrieval**: Optimizing queries and structuring MongoDB collections effectively to handle large datasets, ensuring quick and reliable data retrieval for users searching for accommodations.
- **Real-Time Updates**: Implementing real-time data updates on dorm availability status and user bookings, enhancing the overall user experience and accuracy of displayed information.
- **User Experience (UX) Design**: Balancing a clean design with comprehensive filtering options to ensure users could easily navigate and find relevant dorm options.

## Outcome

The T-PAK Dormitory Matching Platform successfully connects users with dormitory accommodations across Thailand. By integrating search filters and matching algorithms, users can efficiently locate and book suitable dorms. The platform’s architecture allows for future enhancements, such as personalized recommendations and real-time chat support with dorm providers.
